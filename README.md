# Personal Portfolio Website

## 📌 Project Overview
This is a responsive Personal Portfolio Website developed using HTML, CSS, and JavaScript. The website showcases personal information, career objectives, educational background, technical skills, experience, projects, and contact details.

The portfolio serves as an online professional profile where visitors can learn more about the developer and explore their work.

---

## ✨ Features

- Responsive Navigation Bar
- Home Section
- About Section
- Career Objective
- Education Details
- Technical Skills
- Experience Information
- Projects Showcase
- Contact Section
- Resume Download Option
- Dark/Light Theme Toggle

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🚀 How to Run

1. Download or clone the project.
2. Open the project folder.
3. Double-click on `index.html` or open it using any web browser.
4. Explore the portfolio website.

## 📖 Sections Included

### Home
Displays name, role, and quick action buttons.

### About
Provides a brief introduction about the developer.

### Objective
Highlights career goals and aspirations.

### Education
Displays academic qualifications and achievements.

### Skills
Lists technical and professional skills.

### Experience
Shows internship or work experience details.

### Projects
Displays completed projects with descriptions.

### Contact
Provides contact information for communication.
